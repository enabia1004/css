[포지셔닝 익히기]

포지셔닝은 블럭레벨 요소들을 배치하기 위한 속성으로 사용됩니다!

가장 많이 사용되는 속성들은 3가지입니다!

  1. static : 기본값. 일반적인 문서의 흐름을 따름.
  2. relative : 상대적인 위치지정. 원하는 위치에 배치할 수 있으며, 기본적인 문서의 흐름을 따름
  3. absolute : 절대위치지정. 원하는 위치에 배치할 수 있으나, 기본적으로 부모요소의 위치를 기준으로 배치됨
  4. fixed : 블록레벨 요소를 특정위치에 고정할때 사용됨.
  5. z-index -> 공중에 띄어져있는 position끼리 우선순위를 바꿈
